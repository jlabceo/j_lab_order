📲 앱 설치 방법

홈 화면에 추가하면 앱처럼 바로 실행할 수 있어요

1
Chrome 브라우저로 접속 반드시 Chrome 앱으로 주문 링크를 열어주세요
2
하단 배너 또는 메뉴에서 설치 화면 하단에 "앱 설치" 배너가 뜨면 탭하거나,
우측 상단 ⋮ (점 세 개) → "앱 설치" 또는 "홈 화면에 추가"
3
"설치" 버튼 탭 팝업에서 설치를 확인하면 홈 화면에 아이콘이 추가됩니다
💡 이후에는 홈 화면의 J-Lab 주문 아이콘을 탭하면 바로 실행됩니다
1
Safari 브라우저로 접속 ⚠️ 아이폰은 반드시 Safari를 사용해야 합니다
Chrome, 카카오 인앱 브라우저에서는 설치 불가
2
하단 공유 버튼 탭 화면 하단 가운데 □↑ (공유) 버튼을 탭하세요
3
"홈 화면에 추가" 선택 스크롤을 내려 "홈 화면에 추가"를 탭한 후 우측 상단 "추가" 탭
💡 카카오톡에서 링크를 받았다면: 링크 탭 → 우측 하단 Safari로 열기 탭 → 위 절차 진행
